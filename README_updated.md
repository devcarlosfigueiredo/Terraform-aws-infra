> 🚀 Projeto prático de **Infrastructure as Code (IaC)** utilizando Terraform para provisionamento automatizado de infraestrutura na AWS, seguindo boas práticas de **DevOps, modularização e segurança**.

---

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido com foco em simular um ambiente real de infraestrutura em cloud, aplicando conceitos utilizados no dia a dia de times DevOps.

Principais objetivos:
- Automatizar provisionamento de infraestrutura na AWS
- Aplicar boas práticas com Terraform (modularização e reutilização)
- Criar base escalável para ambientes reais (dev/prod)
- Demonstrar habilidades práticas para o mercado de Cloud/DevOps

---

## 🧠 Tecnologias e Conceitos Aplicados

- Terraform (Infrastructure as Code)
- AWS (EC2, VPC, Subnet, IGW, EIP)
- Docker (provisionamento automático via user_data)
- Modularização de código
- Ambientes separados (dev/prod)
- Boas práticas de segurança

## 🚀 Como Executar

```bash
git clone https://github.com/devcarlosfigueiredo/terraform-aws-infra.git
cd terraform-aws-infra

terraform init
terraform plan
terraform apply
```
